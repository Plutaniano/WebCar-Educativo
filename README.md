# WebCar-Educativo

## Instalação

Testado no Python 3.8.
```
$ git clone https://github.com/Plutaniano/WebCar-Educativo.git
$ cd WebCar-Educativo
$ pip install -r requirements.txt
$ python .\server\server.py
```
